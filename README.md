# Abhishek Akkal – Resume Website

This is my personal resume website built using HTML. It showcases my skills, projects, and learning journey as I work towards becoming a Full Stack Developer.

## 🚀 About the Project

This project is a simple and clean resume website created using only HTML. The goal was to structure content properly and present my work in a professional format.

## 🛠️ Tech Stack

* HTML

## 📂 Features

* Clean and structured resume layout
* About Me section
* Skills section
* Projects section with descriptions and links
* Learning Journey section
* Contact links (Email, LinkedIn, GitHub, X)

## 📁 Projects Included

* **Amazon Clone (UI)** – Built using HTML & CSS with responsive layout and structured design
* **Wikipedia Clone (HTML)** – Created using HTML with tables and entities demonstration
